# 16 Monetization

## Monetization Philosophy

Graceward can have a generous free tier while charging for features that create ongoing AI, transcription, storage, and review costs.

## Free Tier Ideas

- Text journaling
- Limited voice reflections
- Limited AI reflections
- Prayer request tracking
- Gratitude tracking
- Basic journal history
- Basic reminders

## Pro Tier Ideas

- Longer voice notes
- More AI reflections
- Weekly reviews
- Monthly reviews
- Long-term memory
- Advanced prayer tracking
- Faithfulness timeline
- More cloud backup
- Advanced RAG-powered reflection
- Export options
- Custom modes

## Server-Side Entitlements

Never trust client for:

- Pro status
- voice duration limit
- AI usage limit
- weekly review access
- monthly review access
- storage limit

## Possible Launch Strategy

Option A: Free beta first, Pro later.

Pros:

- Easier feedback
- Easier product learning
- Less App Store subscription complexity at first

Option B: Subscription from launch.

Pros:

- Validates willingness to pay
- Offsets AI/transcription costs

Recommended:

Start with subscription-ready architecture, but consider free beta first.
